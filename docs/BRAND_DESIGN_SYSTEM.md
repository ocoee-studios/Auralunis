## Logo
The Chronaura logo is a gold crescent "C" within an astrolabe compass:
- **C** = Chronaura (the letter)
- **Crescent moon** = the Moon, cycles, time
- **Sun burst center** = the Sun, time/light
- **Compass rings** = orbits, planets, guidance
- **Astrolabe dial** = exploration, precision, the beauty of the universe
- **Gold spheres** at compass points = celestial bodies

### Logo variations
1. Full wordmark: logo + "CHRONAURA" + "THE INTERACTIVE ASTRAL CLOCK"
2. Compact: logo + "CHRONAURA" (no subtitle)
3. Monochrome: white/silver version for dark backgrounds
4. Icon only: the crescent C compass (used as app icon)

## Design principles
1. **Dark-first** — Cosmic Black background, content floats on glass surfaces
2. **Gold accents** — Astral Gold for interactive elements, never overused
3. **Liquid Glass** — expo-blur BlurView on glass panels, iOS system materials
4. **Star dust** — subtle animated particles on key surfaces
5. **Real data** — no placeholder text, all sky data computed from ephemeris
6. **Generous space** — premium apps breathe, cards have 14-16px padding
7. **Serif titles** — Cinzel for brand moments, Montserrat for everything else
