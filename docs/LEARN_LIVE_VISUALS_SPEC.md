# Chronaura Learn Tab — Live Visuals Spec

## Goal
Replace flat educational pages with beautiful living visual modules.

## Visual modules
- Solar System / Planets: live mini orrery
- Moon: live moon phase sphere
- Constellations: ignition/tracing visual
- Stars: brightness + color comparison
- Deep Sky: glowing object layer
- Milky Way: band + galactic center visual
- 30 Nights: progress/timeline visual

## Production path
Current visuals are lightweight animated React Native modules.
Later they can connect to:
- real ephemeris data
- live moon cycle
- Sky Lens
- Celestial Archive
- saved lesson progress
- interactive quizzes
